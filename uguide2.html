Universal Configurable AC Control Board

User Guide (Draft Version)

This open-source AC control board is designed to replace damaged non-inverter AC PCBs and add WiFi-based smart features.

  • Control Layer: ESP32
  • Power Switching Layer: Relay & SSR
  • User Interface: Web Dashboard & Display

Control Section

  • ESP32 Development Board
  • Display Module
  • IR Receiver (Planned)
  • Buzzer (Optional)

Power Switching Section

  • 5V Relay Module (Compressor)
  • AC-rated SSR (Blower – Pending)

Power Supply Section

  • 230V AC to 5V SMPS Module
  • 3.3V Regulation for ESP32

WARNING: System interfaces with AC mains voltage.

  • Disconnect power before wiring.
  • Ensure relay rating matches compressor load.
  • Maintain isolation between high and low voltage sections.
  • Use protective enclosure.
  • Turn OFF AC mains.
  • Identify compressor control line.
  • Connect relay COM and NO.
  • Power on ESP32 and test via web dashboard.
  • Upload firmware to ESP32.
  • Configure WiFi credentials.
  • Access device IP address in browser.
  • Toggle Compressor ON/OFF
  • Adjust Temperature
  • View Status on Display
  • Blower Speed Integration
  • IR Remote Mapping
  • PCB Version 1.0 Release
